SURFACE WEB

La Surface Web es la parte visible de Internet que puede ser encontrada mediante motores de búsqueda como Google, Bing o Yahoo.

¿Qué es la Surface Web?

La Surface Web representa la pequeña parte de Internet accesible para cualquier usuario mediante navegadores tradicionales. Incluye sitios web públicos, redes sociales, periódicos digitales, blogs y tiendas virtuales.

Características

  • Contenido indexado por buscadores.
  • Acceso libre para cualquier usuario.
  • Uso cotidiano.
  • Información pública.
  • Representa menos del 10% de Internet.

Ejemplos

  • Google
  • YouTube
  • Wikipedia
  • Facebook
  • Instagram
